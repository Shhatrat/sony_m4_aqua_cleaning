# Sony m4 aqua cleaning packages list

Sony M4 has a lot of preinstalled software. 

To uninstall you need [ADB tools](https://developer.android.com/studio/releases/platform-tools.html) and enter to adb shell.

[Tutorial for begginers](https://www.reddit.com/r/Android/comments/6ftg72/want_to_completely_disableuninstall_those_pesky/)

# Warning!

Keyboard, you need install other before you uninstall this!
```
pm uninstall -k    --user 0  com.sonyericsson.textinput.uxp
```
Launchers, you need install other before you uninstall this!
```
pm uninstall -k    --user 0  com.sonyericsson.home
pm uninstall -k    --user 0  com.sonymobile.simplehome
```
